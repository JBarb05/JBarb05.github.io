# Mettre en place un environnement de travail collaboratif informatique

## Contexte

Cet apprentissage s’inscrit dans mes projets menés à l’IUT depuis plus de 3 ans.  
J’ai utilisé GitHub comme outil central pour le suivi de version, la collaboration et la structuration du travail, en lien avec Docker, la création de sites web et des scripts Python.

## Réalisation

J’ai mis en place et utilisé un environnement collaboratif basé sur GitHub avec :

- gestion de versions continue sur plusieurs projets ;
- utilisation régulière des branches (`feature`, `fix`, `main`) ;
- intégration de Docker pour faciliter la reproductibilité des environnements ;
- versionnage de code Python (scripts d’automatisation) ;
- versionnage de projets web, y compris ce site/portfolio lui-même.

## Preuves illustratives éventuelles

- historique de commits sur plus de 3 ans ;
- organisation claire des repositories (README, arborescence, branches) ;
- pull requests et fusions de branches sur différents projets ;
- dépôt GitHub du site actuellement présenté.

## Analyse réflexive

### Réussites

- Maîtrise complète de Git/GitHub dans un cadre académique et personnel ;
- Bonne gestion des branches et des fusions ;
- Adoption d’une méthode de travail propre, traçable et collaborative.

### Difficultés

- Au départ : gestion des conflits de merge ;
- Harmonisation des pratiques de commit entre membres d’équipe ;
- Mise en place d’une discipline de versionnage régulière.

### Apprentissage

- Structurer un projet pour faciliter la collaboration ;
- Sécuriser l’évolution du code grâce à l’historique Git ;
- Articuler GitHub avec Docker et Python pour améliorer la reproductibilité et l’automatisation.

### Projection pour la suite

Je souhaite approfondir les spécificités de GitLab (CI/CD, gestion de projet intégrée, workflows d’équipe) afin de comparer les approches et élargir mes compétences en environnements collaboratifs.

## Note

**5/5**

## Bilan

Compétence acquise et consolidée : GitHub est devenu mon outil de référence pour développer, versionner et collaborer efficacement.
